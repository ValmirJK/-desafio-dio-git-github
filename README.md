Desafio de Projeto: Git e GitHub 
📋 Sobre o Desafio
Desafio proposto pela DIO (Digital Innovation One) com o objetivo de praticar os conceitos fundamentais de Git e GitHub, desde a instalação e configuração até o envio de projetos para um repositório remoto.

 Objetivos

Instalar e configurar o Git localmente
Criar e gerenciar repositórios no GitHub
Utilizar os principais comandos do Git
Realizar o versionamento de um projeto
Enviar o projeto para o GitHub
